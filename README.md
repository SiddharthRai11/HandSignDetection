## Environment & Dependencies

| Package         | Version    |
|-----------------|------------|
| Python          | 3.11.7     |
| TensorFlow      | 2.12.1     |
| MediaPipe       | 0.10.18    |
| OpenCV-Python   | 4.10.0.84  |
| CVZone          | 1.6.1      |
| NumPy           | latest     |
